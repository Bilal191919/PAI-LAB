# Task 10 - Medical Center Information Bot

This project is a Flask-based chatbot web application for basic medical center information.

## Features
- Answer appointment-related questions
- Show available departments
- Share doctor names, specialties, and available days
- Provide location and emergency contact info

## Project Structure
- `app.py` - Flask backend and chatbot logic
- `templates/index.html` - frontend markup
- `static/style.css` - frontend styling
- `requirements.txt` - Python dependencies

## Run Instructions
1. Open terminal in this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start app:
   ```bash
   python app.py
   ```
4. Open browser:
   ```
   http://127.0.0.1:5001
   ```

## Notes
This is a rule-based educational chatbot, suitable for lab submission and further improvement.
